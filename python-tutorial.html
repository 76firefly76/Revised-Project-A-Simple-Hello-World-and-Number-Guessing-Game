Step 2: Running the "Hello, World!" Script

Creating the Script

Create a new file called hello.py.

Writing the Code

Open hello.py in your code editor and paste the following code:

print("Hello, World!")

Running the Script

Open your command prompt (Windows) or terminal (macOS/Linux).

Navigate to the directory where you saved hello.py.

Type python hello.py and press Enter.

You should see "Hello, World!" printed on the screen.

Expected Output

Hello, World!

Step 3: A Simple Number Guessing Game

Creating the Script

Create a new file called guess_number.py.

Writing the Code

Open guess_number.py in your code editor and paste the following code:

import random

    secret_number = random.randint(1, 10)
    guess = int(input("Guess a number between 1 and 10: "))

    if guess == secret_number:
        print("You guessed it!")
    else:
        print(f"The number was {secret_number}")

Running the Script

Open your command prompt (Windows) or terminal (macOS/Linux).

Navigate to the directory where you saved guess_number.py.

Type python guess_number.py and press Enter.

You should see a message asking you to guess a number between 1 and 10.
